About Me

I am a computer science researcher specializing in program analysis, Android security, and software engineering. My work focuses on developing innovative approaches to detect and analyze security vulnerabilities in mobile applications, particularly in the Android ecosystem.

Android Security Program Analysis Mobile Computing Software Engineering Malware Detection Static & Dynamic Analysis

Research Interests

My research focuses on developing scalable and efficient program analysis techniques for mobile platforms, with particular emphasis on:

  • Android Application Security: Developing novel techniques to detect malicious behaviors and vulnerabilities in Android applications
  • Hybrid Program Analysis: Combining static and dynamic analysis approaches for more comprehensive security assessment
  • Reflection Analysis: Uncovering and analyzing reflective code patterns in Android applications
  • Graph-Based Detection Methods: Utilizing graph structures to identify malicious network behaviors in mobile applications
  • Scalable Analysis Frameworks: Building efficient and robust tools for analyzing large-scale mobile application ecosystems
